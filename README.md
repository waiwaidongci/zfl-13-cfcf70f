# 潮汐池生态模拟

静态浏览器小游戏，直接打开 `index.html` 即可运行。

玩家通过放置岩缝、海藻、贝群和遮阴来维持潮间带生态稳定。系统包含潮汐循环、物种数量变化、捕食关系、低水位压力和生态稳定度评分。

## 任务挑战模式

点击顶栏右上角的「任务挑战」按钮即可进入挑战选择界面。当前提供三个挑战：

- **稳定守护者**：在第6天结束前，将生态稳定度维持在80以上。
- **贝类平衡师**：在第5天到第8天期间，贝类数量保持在18~28之间。
- **低潮抗压**：在10天内，累计低潮压力事件不超过6次。

进入挑战后，右侧面板顶部会出现挑战进度区，实时展示各目标的完成状态（黄色待完成、绿色已达成、红色失败）。挑战完成或失败时，画布上方会弹出状态提示。

**重置按钮行为**：在挑战中点击「重置」会将潮汐池恢复到该挑战的初始状态（第1天、物种与工具配置归零），而不会退出当前挑战。点击进度区底部的「退出挑战」可回到自由模式。
